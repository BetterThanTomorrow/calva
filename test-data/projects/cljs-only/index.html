Welcome to the ClojureScript browser REPL.

This page hosts your REPL and application evaluation environment. Validate the connection by evaluating (js/alert "Hello CLJS!") in the core.cljs file opened in VS Code by Calva.

In that file you'll find a (comment ...) form (aka a Rich comment block) with some more stuff you can evaluate.
