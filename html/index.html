Welcome to Calva! Please report any issues you find on GitHub. Also feel welcome to chat us up on the Clojurians Slack, we're in #calva-dev.

The REPL prompt is a multiline, Paredit powered, Clojure editor. Some useful shortcuts are:

  • alt+up/down: for history navigation
  • alt+enter: to submit the current line
  • ctrl+enter: to submit the current line and pretty print the results

See also Calva Top 10 Commands.

NB: It is a good habit to start any Calva session by loading some files. Many things do not work before that is done. Use ctrl+alt+c enter in the code editor to load (evaluate) the active file.
