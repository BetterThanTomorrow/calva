The REPL prompt is a multiline, Paredit powered, Clojure editor. Some useful shortcuts are:

  • Alt+Up/Down
  • Navigate the history
  • Alt+Enter
  • Submit the current line
  • Ctrl+D
  • Interrupt a running evaluation
  • Ctrl+N
  • Clear the Edit area
  • Ctrl+L
  • Clear the REPL window

Make it a habit to start any Calva session by loading a file. Use Ctrl+Alt+C Enter in the code editor to load (evaluate) the active file. Many things do not work before that is done.

See also the Calva User Documentation.
